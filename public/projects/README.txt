IMAGES DE PROJETS FREELANCE
───────────────────────────

Place tes deux visuels de projets ici, avec ces noms exacts :

  project-01.jpg   →   Projet 01 (identité + site vitrine)
  project-02.jpg   →   Projet 02 (identité + e-commerce)

FORMAT RECOMMANDÉ
  - Ratio : 4/3 ou 16/9 (paysage)
  - Résolution : 1600 × 1200 px minimum (ou 1920 × 1080)
  - Colorimétrie : noir & blanc ou très désaturé (effect magazine)
    → Dans Next.js, la prop `grayscale` est déjà appliquée sur les images
    → Tu peux fournir des photos couleur, elles seront automatiquement
       converties en N&B à l'affichage
  - Format : JPG optimisé (qualité 85-90)

PLACEHOLDER
  Sans ces fichiers, les zones image affichent un fond gris uni.
  Le site compile et fonctionne même sans eux.
